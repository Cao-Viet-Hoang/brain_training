üìñ How to Play

üéØ Objective

Train your verbal memory and concentration by memorizing and recalling English words.

This game uses words as memory objects - it's NOT about learning word meanings!

‚öôÔ∏è Game Settings

  • Session Rounds: Number of rounds to play (default: 10)
  • Start K: Initial number of words to memorize (default: 5)
  • Max K: Maximum words per round (default: 12)
  • Memorize Time: Base time to study words (decreases each round)
  • Test Time Limit: Maximum time to select words (9 seconds)
  • Distractor: Optional mini-task between memorize and test phases

üéÆ Game Flow

  1. MEMORIZE: Study the displayed words carefully
  2. DISTRACTOR (optional): Complete a quick task to clear short-term memory
  3. TEST: Select ONLY the words you memorized from a mixed list
  4. RESULT: Review your performance and progress to next round

üèÜ Scoring

  • Hit: Correctly selected a memorized word (+12 points)
  • False Alarm: Selected a decoy word (-10 points)
  • Miss: Failed to select a memorized word (-6 points)
  • Speed Bonus: Extra points for fast completion (up to +40)
  • Base Score: Increases with round difficulty

üìà Difficulty Progression

  • Words to Remember: Increases every 2 rounds
  • Memorize Time: Decreases each round
  • Word Difficulty: Progresses from A1 ‚Üí A2 ‚Üí B1 ‚Üí B2
  • Decoy Similarity: Becomes more challenging in later rounds

üí° Tips

  • Focus on word shapes and patterns, not meanings
  • Try to create mental associations between words
  • Don't rush - accuracy matters more than speed
  • Practice regularly to improve verbal memory capacity
